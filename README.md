# first-py-task: worck in requests and Poetry

**Цель:** Отправить GET-запрос к GitHub API, проверить статус и вывести заголовки.

## Инструменты:

- Python 3.12
- Poetry (менеджер зависимостей)
- Библиотека `requests`

## Установка:

#1. Установи Poetry:

    ```bash
    pip install poetry


    ```

#2.Установи зависимости:

    ```bash
    poetry install
    ```

##Запуск:

```bash
    cd src\first_py_task\
    poetry run main.py

```
